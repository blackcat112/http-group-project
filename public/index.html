Custom HTTP/1.1 Server

Welcome! This server was built entirely from scratch using raw Node.js TCP sockets, without relying on high-level HTTP modules.

Available REST API endpoints:

  • GET /dogs
  • GET /dogs/:id
  • POST /dogs
  • PUT /dogs/:id
  • DELETE /dogs/:id

Para servir este archivo más adelante, muévelo a la carpeta pública y configura el enrutador para devolverlo en la ruta estática deseada.
